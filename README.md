# 🗂️ AstrBot Chat Logger Plugin

AstrBot 对话记录与检索插件，支持群聊和私聊的自动记录，并通过 LLM 智能打标签与权重评估。

## ✨ 功能特性

- **自动记录**：监听群聊/私聊消息及 Bot 回复，自动落盘存储
- **LLM 智能标签**：利用配置的 LLM 为每条对话自动提取标签（如 `日常`、`开发`、`闲聊`），支持多标签
- **权重评估**：LLM 自动评估对话重要程度（1-5），便于后续筛选高价值记录
- **批量处理**：异步队列 + 批处理，攒够 10 条或 30 秒超时后统一处理，不影响消息响应速度
- **日志检索**：支持按日期、标签、权重检索对话
- **Markdown 导出**：一键导出格式化的 Markdown 对话记录
- **优雅降级**：LLM 不可用时自动降级为默认标签，不影响核心记录功能

## 📦 安装

在 AstrBot 插件管理中添加本仓库地址，或手动克隆到 `data/plugins/` 目录下。

## 🔧 使用

插件安装后自动启用，无需额外配置。

### 指令

| 指令 | 说明 | 示例 |
| :--- | :--- | :--- |
| `查询日志` | 查询对话日志，支持按日期/标签/权重筛选 | `查询日志 2026-04-24 开发 4` |
| `导出日志` | 导出对话日志为 Markdown 文件 | `导出日志 2026-04-24` |
| `日志标签` | 查看某天的所有标签 | `日志标签 2026-04-24` |

## 📂 数据存储

日志以 JSONL 格式按日期存储在 `data/chat_logs/{session_key}/{YYYY-MM-DD}.jsonl`，每行一条记录：

```json
{
  "time": "14:30:05",
  "sender_id": "12345678",
  "sender_name": "用户名",
  "message": "对话内容",
  "tags": ["开发", "讨论"],
  "weight": 4
}
```

## ⚙️ 依赖

- AstrBot >= 3.4.0
- 已配置可用的 LLM Provider（用于智能标签，不可用时自动降级）
